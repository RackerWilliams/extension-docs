2.1.1.1. List Endpoint Templates

VerbURIDescription
GETv2.0/OS-KSCATALOG/endpointTemplates​?serviceId=string

List Endpoint Templates.

Normal Response Code(s): 200, 203

Error Response Code(s): identityFault (400, 500, …), badRequest (400), unauthorized (401), forbidden (403), badMethod (405), overLimit (413), serviceUnavailable (503), itemNotFound (404)

List Endpoint Templates.

Table 2.2. List Endpoint Templates Request Parameters
NameStyleTypeDescription
X-Auth-TokenheaderString

Required.

serviceIdqueryString

Optional.
