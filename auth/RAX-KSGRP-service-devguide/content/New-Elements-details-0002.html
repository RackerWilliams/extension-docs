2.5.1.1. New Elements

This extension starts returning groups as a part of response when an authenticate call is made. The samples of new elements that are defined as a part of this extension are listed below.

Example 2.1. Groups Sample XML

<groups xmlns="http://docs.rackspace.com/identity/api/ext/RAX-KSGRP/v1.0">
    <group xmlns="http://docs.rackspace.com/identity/api/ext/RAX-KSGRP/v1.0" id="test_global_group_add">
        <description>A Description of the group</description>
    </group>
</groups>

Example 2.2. Groups Sample JSON

{
  "RAX-KSGRP:groups": [
    {
      "id": "test_global_group_add",
      "description": "A description ..."
    }
  ],
  "RAX-KSGRP:groups_links": []
}
