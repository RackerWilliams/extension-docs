2.3.3.2. Get Type Details

VerbURIDescription
GET/{tenantId}/OS-BS/types/{id} List details of the specified type.

Normal Response Code(s): 200, 203

Error Response Code(s): computeFault (400, 500, …), serviceUnavailable (503), unauthorized (401), forbidden (403), badRequest (400), badMethod (405), overLimit (413), itemNotFound (404)

Table 2.28. Get Type Details Request Parameters
NameStyleTypeDescription
tenantIdtemplateString

A unique identifier for the tenant who owns the resource.

idtemplateUuid

A unique identifier for the type.

This operation does not require a request body.
